AIS Wifi help site

In this project, Jet Lam, Jenny Tong, Pholopat Koo, and I , will make the AIS wifi help page more user friendly and accessable for our client Mr. Chiu and Mrs. Smith.

This site is to give information on wifi troubles, i.e logging in.

    Objectives:
  • educate middle school students on how they can make good use of technology to collaborate
  • appeal to middle school students (theme)
  • for middle school students to find answers they have on wifi on this page
  • for middle school students to feel that this site is appealing rather than strict and professional
      • Success criteria:
  • Title
  • Splash page
  • 3 categories (Logging in, Rules and Regulations, Tech Usage Expectations)
  • Include AIS banner "Login info" button that brings users to the corresponding section of the page

      Storyboard for wireframe 1:
    • with Dreamweaver
    • with fireworks

      • I have used both Dreamweaver and Fireworks to create my homepage. Fireworks was more efficient than Dreamweaver in a way that it didn't have too many functions for a first-time user like me to get confused with. Working with the basic layout on fireworks was easy because it let me position where I wanted my images to go. On the other hand, Dreamweaver had a function where I could see the outcome along with the code behind it. This was an helpful tool because I could learn code, at the same time make use of it on Atom (Planningsite).

      From using these two programs, I think Fireworks is useful for creating the layout of the site (wireframes) and adding images because I can create hotspots. Then Dreamweaver can be used to change the CSS, and edit the webpage's look and feel.
